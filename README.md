# AI 前端/全栈开发面试工具集

> 面向 AI 前端开发、AI 全栈开发岗位的面试工具集，包含题库管理、用户管理和个性化面试题生成。

## 📖 核心功能

### 1. 面试题库
- **文件**：[`AI前端开发面试题库.md`](./AI前端开发面试题库.md)
- **当前版本**：v2.6（81 道题）
- **覆盖方向**：AI 场景前端实践、AI 大模型原理、AI Agent 架构、网络协议、TypeScript、React 框架、架构设计、算法与代码、安全防御、工程化监控
- **来源标注**：每道题均标注信息来源（具体文章链接 / 综合整理 / LeetCode 原题），可追溯考证
- **题库来源**：快手、拼多多、高德、字节、阿里、天猫、京东等一线互联网公司真实面经

### 2. 用户管理
- **目录**：[`articles/`](./articles/)
- **功能**：用户简历归档、个性化面试题生成（纯题版 + 含答案版）、面试纪要归档
- **生成指南**：[`articles/面试题生成指南.md`](./articles/面试题生成指南.md)
- **质量评审**：生成后经 Review Agent 独立检查（P0 阻断 / P1 建议 / P2 可选）

### 3. 面经素材库
- **目录**：[`articles/`](./articles/)
- **平台覆盖**：微信公众号、掘金、知乎、小红书、CSDN、技术栈
- **索引文件**：[`articles/_index.md`](./articles/_index.md)

## 🔄 更新机制

- **定时爬取**：每日自动从掘金、知乎、小红书、微信公众号爬取 AI 前端/全栈面经
- **去重 + 补完答案**：新增题目与题库去重，缺答案的自行研究补充
- **来源追溯（强制）**：每道题必须标注来源链接，Review Agent P0 级检查
