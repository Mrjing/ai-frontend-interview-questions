## 逐题深度解析（摘要）

### JWT存储
- localStorage：需手动在请求头携带
- cookie：httpOnly防XSS，自动携带，SameSite防CSRF

### 大文件上传
核心：分片上传（Blob.slice）、断点续传、并发控制、秒传（文件哈希）、合并通知

### SSE连接断开处理
1. 原生重连：EventSource自动重连
2. 自定义重试：指数退避
3. 断点续传：携带Last-Event-ID

### 接口降级方案
1. 超时降级
2. 失败降级（备用接口或缓存数据）
3. 熔断降级（连续失败N次后直接走降级）
4. 限流降级

### LCP
- 最大内容绘制时间，≤2.5s优秀
- 通过PerformanceObserver获取

### Coding题答案

**调度器**：
```javascript
class Scheduler {
  constructor(limit = 2) {
    this.limit = limit
    this.running = 0
    this.queue = []
  }
  add(promiseFactory) {
    return new Promise((resolve, reject) => {
      this.queue.push(() => {
        promiseFactory().then(resolve, reject).finally(() => {
          this.running--
          this.next()
        })
      })
      this.next()
    })
  }
  next() {
    if (this.running < this.limit && this.queue.length) {
      const task = this.queue.shift()
      this.running++
      task()
    }
  }
}
```

**括号生成**：回溯法，时间复杂度O(4^n / √n)

**版本排序**：split('.')按段比较，补0对齐
